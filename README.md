# Monterey

**Monterey** is an iOS application that uses **The Movie Database API**.

- Modular Architecture
- MVVM
- Swift
- UIKit
- Combine
- SnapKit
- Alamofire
- Realm
- Modern Swift Concurrency
- XCTest
